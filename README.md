# Hiten Portfolio

Static portfolio site for Hiten, built with plain HTML, CSS, and JavaScript.

## Live Site

- `https://ahirhiten.github.io/`

## Project Structure

- `index.html`: homepage
- `contact/`: clean contact page route
- `contact.html`: legacy redirect to the clean contact route
- `styles.css`: visual system and layout
- `script.js`: navigation and reveal behavior
- `assets/`: icons, previews, and project visuals
- `404.html`: fallback page
- `site.webmanifest`: web app metadata
- `robots.txt` and `sitemap.xml`: SEO files

## Notes

- The site has no build step.
- The contact form uses Web3Forms with a direct POST submission.
- `.nojekyll` is included so GitHub Pages serves the files as-is.

## Local Development

Open `index.html` directly or serve the folder with any static server.
